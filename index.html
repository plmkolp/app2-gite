Informacion sobre el perfiul de egreso de la especialidad


La informacion que ofrece la carrera de tecnico en programacion permite al egresado, a travez de la articulacion de sabeeres de diversos campos, realizar actividades diriidas a la:

  • Instalacion y desarrollo de software de aplicacion utilizando la programacion estructurada y orientadad a objetos en ambientes web y moviles, con almacenamiento persistente de datos,
  • Asi como la configuracion y administracion de plataforma e-leraning y
  • comercio electronico
Durante el proceso de formacion de los cinco modulos, el estudiante desarrollara o reforzara
Las siguientes competencias profesionales:
  • Desarrolla e instala software de aplicacion utilizando programacion estructurada, con almacenamiento persistente de datos
  • Desarrolla software de aplicacion utilizando programacion estructurada, con almacenamiento persistente de datos
  • Desarrola aplicaciones web y moviles
  • administra sistemas operativos, aplicaciones y servicios
  • Desarrola administra y configura soluciones de e-leraning y comercio electronico

  • modulo I

    Desarrolla software de aplicacion utilizando programacion estructurada, con almacenamiento persistente de datos

  • modulo II

    Desarrolla software de aplicacion utilizando programacion orientada a objetos, con almacenamiento persistente de datos

  • modulo III

    Desarrolla aplicaciones web y moviles

  • modulo IV

    Administra sistemas operativos, de aplicaciones y srvicios

  • modulo V

    Desarrolla adminisra y configura soluciones en e-learning y comercio electronico
